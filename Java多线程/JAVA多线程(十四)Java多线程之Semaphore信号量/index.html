Semaphore这两个构造方法，都必须提供许可的数量，第二个构造方法可以指定是公平模式还是非公平模式，默认非公平模式。

  • 公平模式： 调用 acquire 的顺序就是获取许可证的顺序，遵循 FIFO；
  • 抢占式： 有可能一个新的获取线程恰好在一个许可证释放时得到了这个许可证，而前面还有等待的线程。
